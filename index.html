Welcome to mattsdemo.ax.lt!

Please watch the above video for an overview of the features of this demonstration

You are now on a completely custom server network hosted on Amazon Web Services (AWS). This scalable and portable AWS VPC allows for automated remote deployment of client computers, many of the most popular software for Mac, as well as custom applications and packages. The system can be raplidly replicated to scale an organization's infrastructre to branch offices and remote clients via branch office and client SSL vpn. Active Directory accounts are used across clients and servers (with the exception of linux) in order to authenticate users. Linux administration of users is provided via Chef. Some key projects and systems used in this demonstration include:

  • Amazon Web Services
    • -- VPC
    • -- EC2
    • -- EBS

  • MacOS Client Administration
    • -- Munki
    • -- DeploySudio
    • -- AutoPkg
    • -- MacOS Server
    • -- Active Directory bound

  • Linux Client Deployment and Adminstration
    • -- AWS AMI deployment
    • -- Chef Configuration Management
    • -- Apache Web Server
    • -- OpenSSL Certificate Authority

  • Windows Server and Client Administration
    • -- Active Directory
    • -- DNS
    • -- Group Policy
